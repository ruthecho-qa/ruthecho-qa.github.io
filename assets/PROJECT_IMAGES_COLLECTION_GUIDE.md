## üõ†Ô∏è Step-by-Step Collection Process

### For App Store Screenshots:

1. **Open the app store link** (Google Play or Apple App Store)
2. **Scroll down** to the screenshots section
3. **Right-click on image** ‚Üí "Save image as..." OR
4. **Take a screenshot** of your screen showing the app screenshot
5. **Crop if needed** to remove browser chrome
6. **Compress** using [TinyPNG.com](https://tinypng.com) if over 500KB
7. **Rename** to the correct filename (e.g., `kyzn.jpg`)

### For Website Screenshots:

1. **Open the website** in your browser
2. **Use browser screenshot tool**:
   - Chrome: Right-click ‚Üí Inspect ‚Üí ‚ãÆ menu ‚Üí Capture screenshot
   - Firefox: ‚ãØ menu ‚Üí More tools ‚Üí Take a screenshot
3. **Or use Snipping Tool** (Windows) / Screenshot (Mac)
4. **Crop to desired size** (16:9 ratio)
5. **Compress and rename**
